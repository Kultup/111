# Доступ до адмін панелі

## Створення адмін користувача

### Автоматичне створення (рекомендовано)

1. Перейти в папку `backend`:
```bash
cd backend
```

2. Запустити скрипт створення адміна:
```bash
npm run create-admin
```

### Дані за замовчуванням:

- **Логін:** `admin`
- **Пароль:** `admin123`

### Налаштування через .env

Можна змінити дані адміна через змінні оточення в `backend/.env`:

```env
ADMIN_LOGIN=myadmin
ADMIN_PASSWORD=securepassword123
ADMIN_FIRST_NAME=Іван
ADMIN_LAST_NAME=Петренко
```

Потім запустити:
```bash
npm run create-admin
```

## Вхід в адмін панель

1. Запустити Backend сервер:
```bash
cd backend
npm run dev
```

2. Запустити Admin панель:
```bash
cd admin
npm run dev
```

3. Відкрити браузер: `http://localhost:3000`

4. Ввести дані для входу:
   - Логін: `admin` (або той, який ви вказали)
   - Пароль: `admin123` (або той, який ви вказали)

## ⚠️ ВАЖЛИВО

**Обов'язково змініть пароль після першого входу!**

Для зміни паролю:
1. Увійти в адмін панель
2. Перейти в профіль користувача
3. Змінити пароль

Або через API:
```bash
PUT /api/users/:id
{
  "password": "новий_пароль"
}
```

## Створення додаткових адмінів

Можна створити додаткових адмінів через:
1. API: `POST /api/auth/register` з `role: 'admin'` (потрібен існуючий адмін)
2. Безпосередньо в базі даних MongoDB
3. Модифікувати скрипт `createAdmin.js`
