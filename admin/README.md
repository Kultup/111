# Admin Panel - Навчальна система тестування

## Встановлення

1. Встановити залежності:
```bash
npm install
```

2. Створити файл `.env` на основі `env.example`:
```bash
# Windows
copy env.example .env

# Linux/Mac
cp env.example .env
```

3. Налаштувати API URL в `.env`:
```
VITE_API_URL=http://localhost:5000/api
```

3. Запустити сервер розробки:
```bash
npm run dev
```
