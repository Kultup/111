# Переміщення проєкту для вирішення проблеми з кирилицею

## Проблема
Flutter не може компілювати шейдери, коли шлях містить кирилицю (наприклад, "Новая папка").

## Рішення

Перемістіть проєкт в папку без кирилиці:

1. Створіть нову папку з латинськими символами, наприклад:
   ```
   C:\Users\QwertY\Desktop\learning-system-mobile
   ```

2. Скопіюйте всю папку `mobile` в нову локацію

3. Відкрийте проєкт з нової локації

4. Запустіть:
   ```bash
   flutter clean
   flutter pub get
   flutter run -d P1129S001087
   ```

## Альтернативне рішення (якщо не хочете переміщати)

Можна спробувати використати короткий шлях (8.3 format), але це складніше і не завжди працює.
